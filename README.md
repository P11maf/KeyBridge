# KeyBridge

KeyBridge — local project for managing KeyBridge services and web UI.

## What’s included
- Python backend
- Web templates and static assets

## Quick start
1. Create and activate a Python virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`

> Note: This repo contains sensitive local files (service account keys, local DB). They are included in `.gitignore` to avoid accidental commits. Replace placeholders where needed.
